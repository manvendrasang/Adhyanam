Chase: A completely speech recognizing chatbot

No need to write any texts, just speak your prompts and it will generate the required output.

This chatbot utilizes following python libraries:

1) Uses "datetime" module for functions like to tell time, to greet on the basis of morning, noon and night etc.
2) Uses "os" module for system related functions like locking the desktop, shutting down the system, signing out from the system etc. 
3) Uses "webbrowser" and "wikipedia" for internet functions like searching a topic on wikipedia, opening certain websites, helping with internet related searches etc.
4) Uses "psutil" for utility based functions
5) Uses "speech_recognition" for speech prompts given by the user to analyze them and give outputs.
6) Uses "requests" to make api based requests
7) Uses "pyttsx3" for converting text to speech
